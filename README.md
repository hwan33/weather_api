# 2022.01.21 업데이트

- 4가지의 상세 기능 중 단기 예보 조회 api 구현
- json-simple 라이브러리 활용해 JSON 파싱
- 10개 이상의 카테고리 변수 중에서 일부만 필터링을 거쳐 db에 저장
- JPA, mySQL 사용
- 현재는 좌표 값인 nx, ny를 임의의 수로 둔 상태
- localhost:8080/api/weather에 접속하면 db 쿼리까지 완료

# 2022.02.07 업데이트

# 2022.02.09 업데이트

- 필요 없는 메서드와 파라미터 제거
